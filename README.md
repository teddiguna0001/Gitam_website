✨ Features
Feature	Description
🏫 Responsive Design	Works seamlessly on desktop, tablet, and mobile devices
🎨 Consistent Layout	Same header, navigation, and styling across all pages
📝 Feedback Form	Integrated with Formspree for email delivery
⭐ Star Rating System	Interactive 5-star rating for feedback
📊 Course Listings	UG and PG courses organized by category
💰 Fee Structure	Separate pages for UG and PG fee details
🎓 Scholarship Information	Detailed scholarship slabs and eligibility criteria
🌐 Easy Navigation	Quick access to all pages from navigation menu
✅ Form Validation	Client-side validation with success/error messages
🚀 No Server Required	Works with static hosting (GitHub Pages, Netlify, etc.)
🔄 Auto-Reset Form	Form resets automatically after successful submission
📱 Mobile-Friendly	Optimized for all screen sizes
